# Sea Battle - Python Implementation
Una versione robusta della Battaglia Navale focalizzata sulla pulizia del codice e sulla gestione della logica di gioco via terminale.

## Caratteristiche Tecniche
- **Matrix Representation**: Utilizzo di liste annidate (Nested Lists) per rappresentare la griglia di gioco 10x10.
- **Input Validation**: Sistema di controllo per prevenire inserimenti errati (coordinate fuori range o già colpite) tramite gestione delle eccezioni.
- **Randomization**: Posizionamento automatico delle navi della CPU utilizzando il modulo `random`.
- **Game Loop**: Gestione strutturata dei turni con verifica immediata delle condizioni di vittoria (tutte le navi affondate).

## Come Eseguire
Assicurati di avere Python 3 installato.
1. Clona il repository: `git clone https://github.com/GabriByte404/battaglia-navale-python.git`
2. Avvia il gioco: `python3 main.py`
